# 🧸 React のライフサイクルまとめ

## 💡 ライフサイクルとは？

React コンポーネントが「**生まれて、更新されて、消える**」までの流れのこと。

---

## 🪴 ライフサイクルの 3 段階

| フェーズ         | 内容                                             |
| ---------------- | ------------------------------------------------ |
| **マウント**     | 初めて画面に表示される瞬間（初期化）             |
| **更新**         | 状態（state）や props が変わって再表示されるとき |
| **アンマウント** | 画面から消えるとき（終了・破棄）                 |

---

## 🧠 ライフサイクルと `useEffect`

React では、`useEffect()` を使ってこの流れに合わせた処理を書く！

### 🌱 マウント時に実行（初回のみ）

```js
useEffect(() => {
  console.log("🌱 マウントされました！");
}, []);
```
