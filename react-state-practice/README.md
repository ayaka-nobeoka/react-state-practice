## 🧠 React の考え方まとめ（超重要）

| キーワード             | 意味                                      |
| ---------------------- | ----------------------------------------- |
| **状態（state）**      | ユーザー操作などで変わる「覚えておく値」  |
| **再レンダリング**     | state が変わると画面が自動で書き換わる    |
| **useState()**         | React で state を作る特別な関数（フック） |
| **関数コンポーネント** | 見た目 + ロジックを 1 つにまとめる部品    |

- `useState` は React の「記憶」機能
- 値が変わると再描画（再レンダリング）される
