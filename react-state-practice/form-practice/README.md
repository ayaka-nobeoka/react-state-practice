# 📝 React フォーム練習：名前入力アプリ

## 🎯 目標

React の state を使って、フォームに入力した名前を画面に表示する簡単なアプリを作成します。

---

## 🔧 機能概要

- 名前を入力できるテキストフォーム
- 入力された名前を即時に表示
- （オプション）空欄なら「ゲストさん」と表示
- （オプション）リセットボタンで入力を初期化

---

## 🧠 学習ポイント

| キーワード       | 説明                                     |
| ---------------- | ---------------------------------------- |
| `useState`       | 入力値を保存するための state を作る      |
| `onChange`       | 入力フォームの値の変化を検知するイベント |
| `value`          | フォームに表示する現在の入力値           |
| `e.target.value` | 入力された文字列を取得する方法           |

---

## 🪛 実装ステップ（ヒント）

1. `useState()` で名前を保存する変数を作る
   👉 `const [name, setName] = useState('');`

2. `<input>` を表示し、`value` と `onChange` を使って連動させる
   👉 `onChange={e => setName(e.target.value)}`

3. 名前を画面に表示する
   👉 `<p>Hi, {name}さん！</p>`

4. オプション：入力が空のとき「ゲストさん」と表示する

5. オプション：リセットボタンを追加
   👉 `setName('')` を使って初期化
